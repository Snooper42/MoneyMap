MoneyMap v0.1.10 — Dashboard Command Center Redesign + Accounts Premium UI
Released: 2026-05-30

MAJOR CHANGES

Dashboard (view-overview)
  - Complete visual redesign of the command center
  - New full-width command band: prominent next-action headline, 4 answer buttons
    (What changed / Where is money going / Am I on track / Upcoming charges), quick action row
  - KPI strip moved to right side of band: Net worth, Cash flow, Review queue
  - Quick actions bar: 6 compact actions (Import, Add, Review, Accounts, Snapshot, Backup)
  - Main 2-column layout: Cash flow chart + Spending categories on left;
    Budget pressure + Review status + Accounts summary on right
  - 4-column bottom grid: Recent activity, Upcoming recurring, Goals, Debt snapshot
  - Canonical renderer: src/js/ui/dashboard-v0.1.10.js
  - Canonical CSS: src/css/dashboard-v0.1.10.css

Accounts (view-accounts)
  - Premium page header with gradient, large net worth number, delta label
  - Compact category filter pill bar with per-category icons and totals
  - Clear Assets vs Liabilities side headers with totals
  - Account rows grouped by category section (group card with icon + metadata)
  - More compact account rows with colored category badges
  - Right-side summary card: allocation bar, category breakdown list, action buttons
  - Canonical renderer: src/js/ui/accounts-v0.1.10.js
  - Canonical CSS: src/css/accounts-v0.1.10.css
