MoneyMap v0.1.9

Focus: Dashboard and Accounts polish before any framework migration.

Highlights
- Dashboard now has a tighter decision-board layer that answers what changed, where money is going, whether the user is on track, and what upcoming charges are detected.
- Accounts now has a labeled filter panel, reset control, clearer category grouping, tighter rows, improved summary metadata, and Snapshots naming.
- Favicons were simplified into transparent orange M icons for light and dark mode.
- Local preview launchers now check for index.html, prefer Python when available, and fall back to port 8081 if 8080 is busy.

Compatibility
- Static GitHub Pages compatible.
- No backend, cloud sync, analytics, or external API calls.
- Existing localStorage data shape is preserved.
