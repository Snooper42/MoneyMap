MoneyMap v0.1.6

Desktop command-center release.

Major changes:
- New desktop Overview focused on next action, net worth, cash flow, review status, budget pressure, spending categories, accounts, goals, debt, recurring charges, and recent activity.
- Removed duplicate dashboard renderer references from index.html.
- Fixed duplicate final build guard reference.
- Stable chart sizing for net-worth canvases during hover, click, scroll, and resize.
- New light/dark favicon using a sideways MoneyMap M with a dollar-mark concept.
- Canonical build and cache token bumped to v0.1.6.

Still static and local-first:
- No backend.
- No external APIs.
- No analytics.
- Existing localStorage key and data shape preserved.
