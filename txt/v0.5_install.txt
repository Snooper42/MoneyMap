MoneyMap v0.5 install notes

1. Back up the existing MoneyMap folder or branch before replacing files.
2. Copy the changed files into the deployed MoneyMap directory, preserving folders.
3. Confirm these new files exist:
   - src/js/ux-v0.5.js
   - src/css/ux-v0.5.css
   - docs/PATCH_NOTES_v0.5.md
   - txt/v0.5_install.txt
4. Open the app with a cache-busting URL once:
   index.html?mmcache=v0.5
5. Hard refresh the browser.
6. Open Settings and verify the release/build label reads v0.5.
7. On mobile width, verify bottom navigation shows Home, Txns, Review, Accounts, and More.
8. Open More and confirm Import CSV, Budgets, Net worth, Investments, Subscriptions, Debt, Credit, Goals, Rules, and Settings are reachable.

Local data note
- v0.5 keeps the existing localStorage key, moneymap_v1.
- Existing user data should remain available after replacing files.
- Export a backup JSON before deploying or testing reset/import flows.

Rollback
- Restore the previous v0.4 files if needed.
- Browser data does not need migration for rollback because the storage key and data shape were not changed.
