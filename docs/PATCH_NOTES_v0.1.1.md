# MoneyMap v0.1.1 Patch Notes

## Dashboard
- Replaced the mobile and desktop overview with a chart-first dashboard.
- Added a Review-first shortcut rail at the top of the dashboard.
- Added cash-flow trend bars, net-worth sparkline, spend mix bars, budget pressure, bills watch, review health, and recent activity.
- Hid legacy overview panels on the Overview screen to remove duplicated sections and mobile blank space.

## Transaction editor
- Reworked the transaction drawer for mobile and desktop.
- Added a live transaction preview, cleaner Spend/Income segmented control, quick category chips, transfer hiding, and sticky actions.
- Rounded edited amounts to cents so imported floating point values do not show long decimals.

## GitHub Pages cache/build label
- Bumped the build to v0.1.1.
- Added a final build guard so legacy UX layers can no longer relabel the current app as v0.8.
- Updated cache-busting query strings for all assets.
