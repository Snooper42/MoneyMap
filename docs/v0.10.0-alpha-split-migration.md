# MoneyMap v0.10.0-alpha Split Migration Notes

## Scope

Continued the safe migration from the large extracted `app.js` and `styles.css` files into smaller static assets.

## JavaScript split

Created these modules and updated `index.html` load order:

- `src/js/utils.js`
- `src/js/state.js`
- `src/js/storage.js`
- `src/js/navigation.js`
- `src/js/accounts.js`
- `src/js/transactions.js`
- `src/js/budgets.js`
- `src/js/investments.js`
- `src/js/charts.js`
- `src/js/settings.js`
- `src/js/app.js`

`app.js` now keeps bootstrap logic, report template code, demo-data helpers, and the later patch layers. The generated report/print template was intentionally left intact to reduce break risk.

## CSS split

Created these files and preserved the original cascade order:

- `src/css/base.css`
- `src/css/layout.css`
- `src/css/components.css`
- `src/css/mobile.css`

The CSS split is intentionally conservative. It keeps contiguous source order rather than aggressively reclassifying every selector.

## Compatibility

- Static-file only
- GitHub Pages compatible
- No backend added
- Local browser storage retained
- No product features added
