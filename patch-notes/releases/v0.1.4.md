# MoneyMap v0.1.4

---

## Changes

- Removed the external-style action from net-worth dot popovers.
- Kept chart point details local-only with account, asset, liability, and change rows.
- Hardened the Accounts page against older account-renderer races that could show a new build label without the matching desktop changes.
- Added clickable chart-point fallback behavior to the older Accounts chart path.
- Added clickable net-worth chart dots with persistent popovers.
- Rebuilt the desktop Accounts page around a chart-first performance card, grouped account list, and summary panel.
- Added custom selectable account icons and saved the selected icon with each account.
- Updated the canonical build label and cache token to `v0.1.4`.
