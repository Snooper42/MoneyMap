# MoneyMap v0.1.8

Focus: Dashboard and Accounts polish before framework migration.

---

## Changes

- Tightened the Dashboard money brief with top spending, budget status, net worth, next action, cash flow, and review status.
- Added Dashboard account chips that open Accounts with the matching category filter.
- Added Accounts filters for all groups, assets, debt, and individual real-world categories.
- Tightened Accounts row density, spacing, summary category rows, and active category state.
- Preserved the existing localStorage account structure while mapping existing account types into the expanded taxonomy.
- Replaced favicon SVGs with cleaner orange `M` icons for light and dark mode.
- Added `START_WINDOWS_PREVIEW.bat` and `START_MAC_PREVIEW.command` to the flat preview package.
- Updated the canonical build/cache token to `v0.1.8`.

---

## Privacy

- Static GitHub Pages compatible.
- No backend.
- No analytics.
- No external APIs.
- No change to existing localStorage data shape.
