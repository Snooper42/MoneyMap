# MoneyMap v0.1.9

Focus: Dashboard and Accounts polish before framework migration.

---

## Highlights

- Dashboard refined into a decision board focused on what changed, where money is going, whether the user is on track, and upcoming charges.
- Dashboard spacing, card hierarchy, quick actions, and command-center behavior tightened.
- Accounts gained a labeled filter panel, reset control, clearer category grouping, tighter rows, improved summary metadata, and Snapshot naming.
- Account summary/sidebar clarified active groups, included accounts, and asset/debt totals on desktop.
- Favicon simplified to transparent orange `M` variants for light and dark mode.
- Local preview launchers hardened with `index.html` checks, Python detection, and fallback from port 8080 to 8081.
- Canonical build/cache token bumped to `v0.1.9`.
- Existing localStorage data shape preserved.

---

## Compatibility

- Static GitHub Pages compatible.
- No backend.
- No cloud sync.
- No analytics.
- No external API calls.
