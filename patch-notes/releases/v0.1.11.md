# MoneyMap v0.1.11

Date: 2026-05-30

---

## What changed

Browser back/forward navigation now works.

Every time the user switches views, a history entry is pushed so the browser back and forward buttons navigate between views. The URL hash also updates, so reloading or bookmarking a URL can land on the intended view.

Implemented through:

```text
src/js/core/router-v0.1.11.js
```

---

## Also fixed

- Accounts dot-click popover CSS restored after prior legacy cleanup.
- Accounts grouping updated so checking, savings, money market, and CDs appear under one Cash category.

## Compatibility

- Static file compatible.
- GitHub Pages compatible.
- No server configuration required.
- Existing localStorage data preserved.
