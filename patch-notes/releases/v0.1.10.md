# MoneyMap v0.1.10

Date: 2026-05-30  
Focus: Dashboard command center redesign and Accounts premium UI

---

## Dashboard

- Complete visual redesign of the command center.
- New full-width command band.
- Prominent next-action headline.
- Four answer buttons: What changed, Where is money going, Am I on track, Upcoming charges.
- KPI strip for net worth, cash flow, and review queue.
- Quick actions for Import, Add, Review, Accounts, Snapshot, and Backup.
- Two-column main layout with cash flow, spending categories, budget pressure, review status, and account summary.
- Bottom grid for recent activity, upcoming recurring charges, goals, and debt.
- Canonical renderer: `src/js/ui/dashboard-v0.1.10.js`.
- Canonical CSS: `src/css/dashboard-v0.1.10.css`.

---

## Accounts

- Premium page header with gradient treatment, large net-worth value, and delta label.
- Compact category filter pill bar with icons and totals.
- Clear assets vs liabilities section headers.
- Account rows grouped by category sections.
- More compact rows with colored category badges.
- Right-side summary card with allocation bar, category breakdown, and action buttons.
- Canonical renderer: `src/js/ui/accounts-v0.1.10.js`.
- Canonical CSS: `src/css/accounts-v0.1.10.css`.
